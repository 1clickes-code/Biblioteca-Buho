01

Del concepto a la herramienta

En el 0.2 aprendiste que el contexto es lo que separa un prompt genérico de uno que funciona. Que la calidad importa más que la cantidad. Que construir con la IA es diferente a simplemente preguntarle.

Todo eso es real. Y ahora viene la parte práctica: hay una herramienta gratuita y de código abierto que automatiza exactamente eso. No lo hace por ti — lo hace contigo, que es la diferencia que ya conoces.

La filosofía de Prompt Master

"El mejor prompt no es el más largo. Es aquel en que cada palabra tiene peso." Eso es eficiencia de tokens aplicada desde el primer intento.

Se llama Prompt Master. Funciona como una skill de Claude — una extensión que vive en tu entorno y trabaja contigo dentro de las conversaciones. En los próximos minutos la tendrás instalada.

02

Qué es Prompt Master y qué puede hacer

github.com/nidhinjs/prompt-master

Prompt Master es una skill open source creada por la comunidad. Su función es una sola: construir prompts precisos desde el primer intento, sin iterar a ciegas.

Cómo funciona
1 Detecta la herramienta destino — Claude, ChatGPT, Cursor, Gemini, Midjourney, o más de 30 plataformas. Adapta el prompt al contexto de cada una.
2 Extrae tu intención en 9 dimensiones: rol, objetivo, contexto, formato, tono, restricciones y más.
3 Hace hasta 3 preguntas de aclaración — las mínimas necesarias para llegar a algo preciso sin hacerte perder tiempo.
4 Aplica técnicas probadas: asignación de rol, ejemplos few-shot, etiquetas XML, anclajes de precisión y razonamiento encadenado.
5 Audita la eficiencia de tokens antes de entregarte el prompt final — sin palabras de relleno, sin instrucciones redundantes.
Nota importante

Prompt Master está construido para Claude Code y claude.ai, pero los prompts que genera funcionan en cualquier modelo. La herramienta es específica — los resultados son universales.

03

Cómo abrir la terminal en Claude Code

Para instalar Prompt Master necesitas una terminal. En Claude Code es más directo de lo que parece.

Abre la terminal
→ En VS Code con extensión Claude Code: atajo Ctrl + ` (Windows/Linux) o Cmd + ` (Mac). También desde el menú: Terminal → New Terminal.
→ En la app de escritorio Claude Code: busca el icono de terminal en la barra lateral o usa el mismo atajo Ctrl/Cmd + `.
→ Sin Claude Code: abre tu terminal del sistema directamente — Terminal.app en Mac, o la que uses en tu sistema. El comando funciona igual.

Una vez abierta la terminal, ya puedes pasar a instalar la skill. Un solo comando.

04

Instalación — global o local

Antes de copiar el comando, hay una decisión pequeña pero importante: instalar la skill de forma global o local. Afecta a cuándo y dónde estará disponible.

Global Local (proyecto)
Dónde se instala ~/.claude/skills/ .claude/skills/ del proyecto actual
Disponibilidad En todos tus proyectos y conversaciones Solo en el proyecto donde la instalas
Ventaja Instalar una vez y olvidarte Aislado — no interfiere con el resto
Inconveniente Activa siempre, aunque no la necesites Hay que instalarla en cada proyecto
Recomendado si... Vas a usarla habitualmente Quieres probarla antes de comprometerte

Mi recomendación: empieza con global. Si llegas hasta este recurso, ya sabes que la vas a usar.

Instalación global — disponible en todos tus proyectos
git clone https://github.com/nidhinjs/prompt-master.git ~/.claude/skills/prompt-master
Instalación local — solo en el proyecto actual
git clone https://github.com/nidhinjs/prompt-master.git .claude/skills/prompt-master
Verificación

Una vez ejecutado el comando, reinicia Claude Code. En la siguiente conversación Prompt Master ya estará activo y listo para usar.

05

Pruébala ahora — una tarea real antes de cerrar

No mañana. Antes de salir de aquí. Elige algo que tengas pendiente — un email, una tarea, un problema que necesita pensarse — y pídele a Prompt Master que construya el prompt contigo.

Hazlo ahora
1. Abre una conversación nueva en Claude Code.
2. Escribe algo como: "Quiero crear un prompt para [tarea concreta]. Ayúdame a construirlo."
3. Prompt Master te hará hasta 3 preguntas. Respóndelas con honestidad — cuanto más concreto, mejor el resultado.
4. Recibe el prompt optimizado. Cópialo y úsalo directamente con la herramienta que necesites.
5. Compara el resultado con lo que obtenías antes de tener Prompt Master. La diferencia es el punto de partida para el Nivel 1.
